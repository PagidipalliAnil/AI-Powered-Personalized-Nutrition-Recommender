# AI-Powered-Personalized-Nutrition-Recommender

An AI-powered nutrition recommendation system using Streamlit, OCR, BMI analysis, and personalized food recommendations.

## Features
- BMI Calculator
- OCR Medical Report Reading
- Disease Detection
- Personalized Food Recommendation
- Dashboard Visualization
- Nutrition Assistant Chatbot

## Tech Stack
- Python
- Streamlit
- Pandas
- OCR
- Machine Learning
